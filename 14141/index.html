3.1 - Раздел диска 10 ГБ (Управление дисками)
Создать новый раздел диска объемом 10 ГБ.
1. Нажмите Win + X и выберите «Управление дисками» (Disk Management).
2. Нажмите правой кнопкой мыши (ПКМ) по диску C: и выберите «Сжать том...» (Shrink Volume...).
3. В поле ввода размера сжатия введите 10240 (в МБ) и нажмите кнопку «Сжать» (Shrink).
4. Нажмите ПКМ по появившейся внизу черной области и выберите «Создать простой том...» (New Simple Volume...).
5. Нажимайте «Далее», выберите букву, файловую систему NTFS, оставьте галку на «Быстрое форматирование» и нажмите «Готово».
3.2 - Режим сна через 5 мин
Установить переход компьютера в спящий режим через 5 минут простоя.
1. Откройте «Параметры» (Settings, клавиши Win + I).
2. Перейдите в раздел «Система» (System) -> «Питание и спящий режим» (Power & sleep).
3. В выпадающем меню спящего режима (Sleep) при работе от сети выберите «5 минут».
3.3 - Создать админа "eksamens"
Создать пользователя "eksamens" с правами администратора и паролем User@12345.
1. Нажмите Win + R, введите lusrmgr.msc и нажмите Enter.
2. Зайдите в папку «Пользователи» (Users). Нажмите ПКМ на пустом месте по центру -> «Новый пользователь...» (New User...).
3. Имя пользователя: eksamens. Пароль: User@12345. Снимите галку «Требовать смену пароля» и поставьте галку «Пароль не ограничен по времени».
4. Нажмите «Создать» и «Закрыть».
5. Кликните дважды по созданному пользователю eksamens -> перейдите во вкладку «Членство в группах» (Member Of).
6. Нажмите «Добавить...» -> введите Administrators (или Администраторы) -> нажмите «Проверить имена» -> «ОК».
7. Кликните на группу Users и нажмите «Удалить», чтобы осталась только группа администраторов. Примените и нажмите «ОК».
3.4 - Создать группу "eksadmin"
Создать группу локальных пользователей "eksadmin" с правами как у Администратора.
1. В открытом окне lusrmgr.msc кликните на папку «Группы» (Groups) слева.
2. Нажмите ПКМ по пустой области в центре -> «Создать группу...» (New Group...).
3. Имя группы: eksadmin. Нажмите кнопку «Добавить...» (Add...).
4. Введите имя пользователя Administrator (или Администратор), нажмите «Проверить имена» и нажмите «ОК».
5. Нажмите кнопку «Создать», затем «Закрыть».
3.5 - Firefox: настройки паролей/загрузок
Отключить сохранение паролей и включить запрос пути сохранения загрузок в Firefox.
1. Откройте Firefox, нажмите на меню (три полоски справа вверху) -> «Настройки» (Settings).
2. Слева перейдите в «Приватность и защита» (Privacy & Security). Прокрутите до блока «Логины и пароли» и снимите галочку с «Предлагать сохранение логинов и паролей».
3. Слева переключитесь на вкладку «Основные» (General). Прокрутите до раздела «Загрузки» (Downloads) и включите галку «Всегда спрашивать, куда сохранять файлы».
3.6 - Сканер: формат JPG по умолчанию
Установить сканер и задать JPG основным форматом сохранения сканов.
1. Подключите сканер. Откройте «Параметры» -> «Устройства» -> «Принтеры и сканеры» -> нажмите «Добавить...».
2. Запустите стандартную программу «Факсы и сканирование Windows» (через поиск Пуск) или программу сканера.
3. Нажмите кнопку «Новое сканирование». В пункте «Формат файла» (File type) выберите JPG (JPEG).
3.7 - Настроить язык интерфейса системы (Latviešu)
Установить язык интерфейса операционной системы на латышский.
1. Откройте «Параметры» (Win + I) -> раздел «Время и язык» (Time & Language) -> вкладка «Язык» (Language).
2. В блоке «Предпочитаемые языки» нажмите «Добавить язык» (+ Add a language).
3. Найдите в поиске Latviešu, выберите его, нажмите «Далее».
4. Установите галочку на «Установить как язык интерфейса Windows» (Set as my Windows display language) и нажмите «Установить».
5. Система предложит выполнить выход (Sign out) — согласитесь для применения языка.
3.8 - Сетевой адаптер: статический IP (WAN)
Настроить IP: 80.152.170.9, Mask: /30 (255.255.255.252), GW: 80.152.170.1, DNS: 91.198.156.20, 194.8.2.2.
1. Нажмите Win + R, введите ncpa.cpl и нажмите Enter (Сетевые подключения).
2. Нажмите ПКМ по адаптеру Ethernet -> выберите «Свойства» (Properties).
3. Кликните дважды по пункту «IP версии 4 (TCP/IPv4)».
4. Выберите переключатель «Использовать следующий IP-адрес» и укажите:
   - IP-адрес: 80.152.170.9
   - Маска подсети: 255.255.255.252
   - Основной шлюз: 80.152.170.1
5. Выберите переключатель «Использовать следующие адреса DNS-серверов» и укажите:
   - Предпочитаемый DNS-сервер: 91.198.156.20
   - Альтернативный DNS-сервер: 194.8.2.2
6. Нажмите «ОК» в обоих окнах.
3.9 - Брандмауэр: Блокировка HTTP (Порт 80)
Заблокировать исходящий порт 80 (HTTP) в брандмауэре.
1. В поиске Windows введите «Монитор брандмауэра защитника Windows» (или wf.msc через Win+R) и откройте его.
2. Слева кликните на «Правила для исходящего подключения» (Outbound Rules).
3. Справа нажмите «Создать правило...» (New Rule...).
4. Тип правила: «Для порта» (Port) -> далее.
5. Протокол: TCP. В поле «Определенные удаленные порты» (Specific remote ports) введите 80 -> далее.
6. Действие: Выберите «Блокировать подключение» -> далее -> далее.
7. Имя правила: введите Block HTTP Out и нажмите «Готово».
3.10 - Визуальные эффекты (Макс. быстродействие)
Выключить все визуальные эффекты Windows для лучшей производительности.

4. Во вкладке «Визуальные эффекты» выберите пункт «Обеспечить наилучшее быстродействие».
5. Нажмите «Применить» и «ОК» во всех окнах.
